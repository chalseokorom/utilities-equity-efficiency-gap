# Electricity Inefficiency & Consumer Rates Analysis

**Does system-level waste correlate with higher costs for residential consumers?**

## Overview

This project investigates the relationship between utility operational inefficiencies—specifically **energy losses** and **poor load factors**—and the **retail rates** charged to residential consumers. Using data from the [CORGIS Electricity dataset](https://corgis-edu.github.io/corgis/python/electricity/), the analysis focuses on identifying whether consumers pay a "premium" for utilities that fail to optimize their energy flow and system demand.
    

## Technologies Used

-   **Python** (Pandas, NumPy)
    
-   **Data Visualization**: Plotly (Heatmaps, Interactive Scatter Plots, Sankey Diagrams)
    
-   **Analysis**: Correlation matrices and OLS Trendline regression
